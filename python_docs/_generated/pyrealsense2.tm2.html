pyrealsense2.tm2¶

class pyrealsense2.tm2¶

The tm2 class is an interface for T2XX devices, such as T265. For T265, it provides RS2_STREAM_FISHEYE(2), RS2_STREAM_GYRO, RS2_STREAM_ACCEL, and RS2_STREAM_POSE streams, and contains the following sensors: -pose_sensor: map and relocalization functions. -wheel_odometer: input for odometry data.
