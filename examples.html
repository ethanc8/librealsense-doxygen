Save to Disk
Demonstrate how to acquire and save video frames on a headless system without graphical user interface (GUI). Individual frames + metadata are save as PNG files.

The multicam sample demonstrates the ability to use the SDK for streaming and rendering from multiple RealSense devices
Point-cloud
Showcase Projection API while generating and rendering 3D pointcloud.
Align
By default, different streams are not necessarily spatially aligned. This example shows how to align depth and color and filter the resulting RGBD image based on depth
Post-Processing
Demonstrating usage of post processing filters for depth images. The SDK offers several filters designed to improve the quality of depth from stereo at reasonable performance
Playback & Record
This examples shows the user how to record RealSense data to file (.bag) for later use and how to play such file back into the application
Software Device
Shows how to create a custom rs2::device. This can be useful for development of converters from / to RealSense and integration with 3rd party devices.
Sensor Control
A tutorial for using the rs2::sensor API. Sensor API is designed to offer low-level access to the hardware, mapping more closely to underlying OS APIs
